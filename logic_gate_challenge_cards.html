üìù Design Requirements for All Challenges

What Students Must Create:

  • Circuit Diagram: Draw the logic gates with inputs and outputs clearly labeled
  • Truth Table: Complete table showing all input combinations and outputs
  • Game Design Document: Explain the game scenario and how the PLD works
  • Physical Model (Optional): Use cardboard, LEDs, buttons, or craft materials to simulate the device
  • Code Pseudocode: Write how the micro:bit would be programmed (doesn't need to be actual code)
  • Presentation: Be ready to explain your PLD design to the class
